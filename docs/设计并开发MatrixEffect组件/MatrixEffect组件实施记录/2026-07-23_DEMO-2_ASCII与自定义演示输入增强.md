# 工作包 DEMO-2：ASCII 与自定义演示输入增强

## 基本信息

- 日期：2026-07-23
- 状态：部分完成
- 对应执行计划工作包：`DEMO-2 ASCII 与自定义演示输入增强`

## 本次实现内容

- ASCII 字符集输入框初始值改为组件默认字符集 `" .:-=+*#%@"`，新增组件默认、精细层次、块元素和圆点四套 Demo 级预设，并保留直接编辑任意字符串的能力。
- ASCII 与自定义场景新增共享输入源控件，支持流动光团、呼吸圆环、流动波浪、静态星旋和本地图片。
- 新增三个模块级稳定动态 Source；全部通过 `MatrixProceduralSource.time` 推进，不创建额外 rAF、定时器或 React 逐帧状态。
- 动态场提供 `playing` 开关，静态示例与上传图片自动禁用；核心仍负责 Reduced Motion、离屏暂停和帧率门控。
- 上传图片直接把 `File` 交给图片 Source adapter；Demo 不上传、不持久化、不创建 object URL，并显示当前文件名。
- ASCII 默认使用流动光团，自定义管线默认使用呼吸圆环，避免两个场景继续显示同一静态构图。
