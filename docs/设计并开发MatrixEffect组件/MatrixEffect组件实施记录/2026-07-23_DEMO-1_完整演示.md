## 本次实现内容

- 新增 `MatrixEffectDemo`，使用圆点、ASCII、自定义三个标签视图；只挂载当前活动场景，避免不可见的动态 Canvas 持续占用资源。
- 圆点场景使用内置柔和光团 Source，提供响应式单元格尺寸、最大半径、速度、对比度和反相控制。
- ASCII 场景使用同源静态图片，提供字符集、响应式单元格尺寸、固定色/源图色和反相控制；字符集输入保留开头空格，不破坏低密度 glyph。
- 自定义场景复用同一个静态图片 Source，通过连续/阈值 Transform 与方块/柱形 Cell Renderer 的独立组合展示核心扩展能力，并提供阈值和单元格尺寸控制。
- Source、Mapper、基础 Levels Transform 使用模块级稳定实例；会变化的 Grid、Transform 数组、Cell 回调和 Renderer 均按最小 primitive 依赖 memo，避免无关控件 render 重建管线。
- 自定义 Cell Renderer 跳过全透明和近零信号格，按 Source Alpha 绘制，不为每格创建对象、Path 或 save/restore。
- Demo 使用稳定 `4:3`/`16:9` 预览比例与桌面双栏、移动端单栏布局；三个 Canvas 均提供非装饰可访问名称和加载失败 fallback，并沿用核心 Reduced Motion 默认策略。
- 新增 1024 x 1024 无损 WebP 示例图：采用固定种子与整数角向系数程序化原创，修复极坐标接缝；同目录 README 记录资产来源和 MIT License。
