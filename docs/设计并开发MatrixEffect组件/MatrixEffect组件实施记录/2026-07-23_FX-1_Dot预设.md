## 本次实现内容

- 新增确定性 `createSoftBlobSource()`：使用 seed 生成 1 到 6 个柔和径向光团，以有效播放时间驱动正弦轨迹和呼吸半径；`speed=0` 时自动成为静态 Source。
- 在不透明基础灰度上叠加白色径向渐变，使不读取 Alpha 的 Luminance Mapper 也能得到连续信号，且透明区域不会因后续反相变为实心图元。
- 新增高性能 `createDotRenderer()`：支持固定色/Source RGB、半径与透明度范围、Source Alpha、同步值曲线、CSS px 坐标和单元格短边半径上限。
- 固定色且透明度恒定时把全不透明圆点合并为单个 Path；每个圆先 `moveTo()` 再 `arc()`，只有发现半透明像素时才执行第二次扫描和逐格绘制。
- 新增 `createCellRenderer()`：每个 Renderer 实例仅创建一个 scratch cell，跨格、跨帧复用；提供中心 UV、原始 RGBA 和 CSS px 几何信息，并拒绝返回 thenable 的异步回调。
- 新增 `DotMatrixEffect` 预设：无 Props 时自动创建柔和光团 Source，固定执行 Luminance -> Invert -> Levels -> additionalTransforms，并把 `backgroundColor` 映射为核心 `clearColor`。
- Source、Renderer、Levels 和 Grid 均按 primitive 字段 memo；grid 按 auto/fixed 分支合并，响应式场景保持 `columns x rows`，fixed 分支不混入 `cellSize`。
- 使用 `forwardRef<MatrixEffectHandle, DotMatrixEffectProps>` 原样转发核心句柄，并从公共入口导出组件、工厂与全部相关类型。
- 复审后补充 `valueCurve` thenable 防护：消费迟到 rejection 后同步抛错，使核心按 Renderer 运行时错误处理，不会把异常帧提交为成功。

## 接口或数据结构变化

- 新增 `SoftBlobSourceOptions` 与 `createSoftBlobSource()`。
- 新增 `CellRendererOptions`、`MatrixRenderCell` 的中心 UV 语义与 `createCellRenderer()`。
- 新增 `DotValueCurve`、`DotRendererOptions` 与 `createDotRenderer()`。
- 新增 `DotMatrixEffectProps` 与 `DotMatrixEffect`；`source` 可省略，且预设不暴露核心的 `renderer`、`mapper`、`transforms`、`clearColor`。
- Dot 默认值为 `color="#71717a"`、`radiusRange=[0.35, 4]`、`opacityRange=[1, 1]`、`cellSize=10`、`cellAspectRatio=1`、`maxCells=10000`、首选 60 FPS。
